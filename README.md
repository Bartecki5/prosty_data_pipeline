# Weather Data Pipeline (Docker + Python + PostgreSQL)

Prosty projekt Data Engineering, który pobiera dane o pogodzie z API Open-Meteo i zapisuje je w bazie PostgreSQL działającej w kontenerze.

## Technologię
* **Python** (requests, psycopg2)
* **Docker & Docker Compose**
* **PostgreSQL**

## Jak uruchomić?
1. `docker compose up -d --build`
